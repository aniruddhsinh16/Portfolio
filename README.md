## 🚀 Quick Start

1. **Clone the repository**
   ```bash
   git clone https://github.com/aniruddhsinh16/projects.git
   cd projects
   ```

2. **Open in browser**
   - Simply open `portfolio.html` in your web browser
   - Or use a local server:
   ```bash
   # Using Python 3
   python -m http.server 8000
   
   # Using Node.js (http-server)
   npx http-server
   ```

3. **Visit the site**
   - Open `http://localhost:8000` in your browser

## 🎨 Customization

### Update Personal Information
Edit `portfolio.html` to add:
- Your name and title
- Professional bio
- Skills and technologies
- Projects and descriptions
- Contact information
- Social media links

### Modify Colors
Update the primary color in `portfolio.css`:
```css
/* Change this color throughout the CSS */
color: #2563eb; /* Current primary color */
```

### Add/Remove Sections
Simply add or remove sections in the HTML:
```html
<section id="section-name" class="section-name">
    <!-- Your content -->
</section>
```

## 🐛 Troubleshooting

### Portfolio not loading?
- Check file paths are correct
- Ensure CSS file is in the same directory
- Clear browser cache (Ctrl+Shift+Delete)

### Animations not working?
- Update your browser
- Check CSS syntax in browser DevTools
- Verify animation duration settings

### Responsive layout broken?
- Test in browser DevTools (F12)
- Check viewport meta tag in HTML
- Verify media queries in CSS
